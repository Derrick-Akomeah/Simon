Press A Key to Start

× Instructions
  • Press the start button or any button the the keyboard to begin
  • A random light will flash. Press it to start the game
  • Another color will flash. Repeat the sequence by pressing the same color
  • The game will continue. After each light sequence that you repeat successfully, a light will be added to the end.
  • When you don't complete a sequence successfully, the Simon game will play a fail sound and you will have to start again. Try to beat your high score!!
